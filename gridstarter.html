CSS Grid without text-align

CSS Grid

I made a class called grid and added properties so the the page will display as a grid. The key property I used is the grid-template-columns property. Now all the boxes with text are in the first column. The boxes with figures are in the second column. The further reading box is in the third column. With the repeat(auto-fit) part of the value, I am able to have all the boxes fit on the page no matter what the screen width is.
